# spren-tui

Placeholder for the Textual-based TUI client that ships with MARSYS Spren v0.4.

The TUI is a four-pane terminal interface (Now / Inbox / Activity / Chat) consuming
the same FastAPI surface as the desktop GUI (SP-019). v0.3 ships only this skeleton
package so the workspace structure and `spren-tui` binary entry point exist; v0.4
fills in the actual Textual app.

## Run (v0.3)

```bash
spren-tui
# → "spren-tui is a placeholder in v0.3 — full implementation lands in v0.4."
```

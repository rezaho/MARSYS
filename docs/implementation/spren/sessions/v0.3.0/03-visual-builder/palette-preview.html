Design system

Palette. Pure warm-white ground. Spren is the only color in the room — peach at the top, soft pink through the middle, magenta toward the base, fading out at the edges. No second accent color anywhere else on the surface. Text is warm near-black; small text is warm gray. The send button is the same warm near-black, becoming magenta on hover.

#FCFAF6 ground
#FFD0AC peach
#FF8FA8 pink
#E85AA0 magenta
#C92E7E deep
#1A1410 ink

Typography. Geist for everything functional. Instrument Serif Italic only when something is personal — your name, Spren's voice, the wordmark, section titles. Two voices: the clean technological frame and the warm italic presence inside it. Geist Mono only for the smallest system text at the bottom (the command-bar hint).

The orb. SVG path clipped to an asymmetric egg (wider at bottom, slightly tapered up). Filled with a radial gradient: peach to pink to magenta, fading transparent at the bottom edge. A turbulence filter overlays grain so the surface is alive, not plastic. Animation is two slow loops — a 12-second vertical drift and a 14-second non-uniform scale + 0.4° rotation — composed so the orb never lands in the same position twice within a normal viewing window. No hard edges anywhere.

Layout. Spren occupies the entire stage. Header is two small marks — the wordmark in italic serif, your avatar in a soft coral-to-magenta gradient. Below the orb: a single welcoming sentence (your name in italic, in magenta) and one subline. Then a single input field. Then a thin command-bar hint at the very bottom. No nav rail. No panels. Nothing competes with the orb.

Motion. Three layers, all slow. The orb floats up and down 8px over 12s. The orb path scales and rotates microscopically over 14s. Hovering the send button colors it magenta over 240ms. Focusing the input draws a soft magenta halo around the field. Everything respects prefers-reduced-motion.

i.

Spren is the interface. Workflows, runs, memory, and settings are surfaces Spren takes you to — reached via ⌘K or by asking. No persistent navigation.

ii.

White is the room. The orb is the warmth in it. Adding any other color outside the orb breaks the meditation.

iii.

Grain is non-negotiable. Without it the orb looks plastic. With it, the orb feels alive.

iv.

Italic serif is reserved. It carries names, voice, and presence. Never a button label, never UI chrome.

If this lands, I lock the design system and dispatch the Researcher with this palette + typography + layout philosophy. If anything is off — orb scale, shape proportions, gradient stops, font weights, copy tone — point at the specific thing.
